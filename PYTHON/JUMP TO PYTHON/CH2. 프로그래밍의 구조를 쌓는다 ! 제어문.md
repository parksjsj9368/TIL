## 2.1 if 문

<br>

- if 문의 기본 구조

if 조건문 1 :

​	수행할 문장 1

elif 조건문 2 :

​	수행할 문장 2

else :

​	수행할 문장3

<br>

<br>

### if문 관련 함수

<br>

| 함수            | 설명                               |
| --------------- | ---------------------------------- |
| x < y           | x가 y보다 작다                     |
| x > y           | x가 y보다 크다                     |
| x == y          | x와 y가 같다                       |
| x != y          | x와 y가 같지 않다                  |
| x <= y          | x가 y보다 작거나 같다              |
| x >= y          | x가 y보다 크거나 같다              |
| x or y          | x와 y 둘 중에 하나만 참이면 참이다 |
| x and y         | x와 y 모두 참이어야 참이다         |
| not x           | x가 거짓이면 참이다                |
| x in 리스트     | 리스트내 x가 있으면 참이다         |
| x not in 리스트 | 리스트내 x가 없으면 참이다         |
